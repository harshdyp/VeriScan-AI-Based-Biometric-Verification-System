# SmartID: Multi-Modal Identity Verification System (Face + OCR + API)

SmartID is a modern, full-stack MVP for multi-modal identity verification. It combines face recognition and OCR-based ID parsing, served via a FastAPI backend and a beautiful Streamlit frontend. The system is designed for rapid prototyping, demo, and educational use, with extensible architecture for future production features.

---

## 🚀 Features
- **Face Verification:** Compare a selfie with an ID photo using state-of-the-art deep learning (facenet-pytorch).
- **OCR Extraction:** Extract text from ID images using Tesseract OCR.
- **Mock Data Validation:** For MVP, validate extracted ID fields against mock data generated by the Faker library.
- **Modern UI:** Intuitive, attractive Streamlit interface with real-time feedback.
- **API-First:** FastAPI backend with clear, documented endpoints.
- **Cross-Platform:** Works on Windows and Linux (with setup notes for both).

## 🛠️ Setup Instructions

### 1. Clone the Repository
```bash
git clone <your-repo-url>
cd ML new
```

### 2. Install Python Dependencies
```bash
pip install -r requirements.txt
```

### 3. Install Tesseract OCR
- **Windows:**
  1. Download the official installer from [UB Mannheim Tesseract Wiki](https://github.com/UB-Mannheim/tesseract/wiki)
  2. Run the installer (default path: `C:\Program Files\Tesseract-OCR\tesseract.exe`)
- **Linux:**
  ```bash
  sudo apt-get update && sudo apt-get install tesseract-ocr
  ```
- **Update the Tesseract path in** `ocr_text_extraction/extract_text.py` **if needed.**

### 4. Run the Backend (API)
```bash
uvicorn api.main:app --reload
```
- The API will be available at [http://localhost:8000](http://localhost:8000)

### 5. Run the Frontend (UI)
```bash
streamlit run frontend/app.py
```
- The UI will be available at [http://localhost:8501](http://localhost:8501)

## 🧪 Usage Guide
1. **Upload your selfie and ID photo** in the UI.
2. **Click "Verify Face"** to check if your selfie matches your ID photo.
3. **Click "Extract and Validate ID"** to extract text from your ID and validate it against mock data.
4. **See instant results** with clear explanations for validation success or failure.

## 🔗 API Endpoints

### `/verify-face` (POST)
- **Description:** Compares a selfie and an ID photo for facial similarity.
- **Request:**
  - `selfie`: image file
  - `id_photo`: image file
- **Response:**
  - `{ "match": true/false }`

### `/extract-id-text` (POST)
- **Description:** Extracts and parses text from an ID image, then validates it against mock data.
- **Request:**
  - `id_image`: image file
- **Response:**
  - `valid`: true/false (do the extracted fields match the mock data?)
  - `reasons`: list of reasons for validation failure (shown in UI)

---

## 🧩 MVP Validation Logic (Faker)
- **Mock Data:** Uses the Faker library to generate a random Name, DOB, Address, and ID number.
- **Parsing:** Extracts these fields from the OCR text using regex.
- **Validation:** Checks if all required fields are present and match the mock data.
- **UI:** Only the validation result and reasons for failure are shown to the user (no raw data).

## 🩺 Troubleshooting
- **TesseractNotFoundError:**
  - Ensure Tesseract is installed and the path in `extract_text.py` is correct for your OS.
  - Test with `tesseract --version` in your terminal.
- **Model Download Issues:**
  - If face recognition fails to load, delete the corrupted model file in `~/.cache/torch/checkpoints/` and restart the backend.
- **API Connection Errors:**
  - Make sure both backend and frontend are running and on the correct ports.
- **OCR Extraction Issues:**
  - Ensure your ID image is clear and text is legible.
